MSC World Explora - VR Experience Storyboard

This storyboard outlines the user journey for the MSC World Explora Virtual Reality experience. The application guides potential clients through a gamified tour of the ship's key amenities, encouraging engagement through a scavenger hunt mechanic.

1. Landing Page (Home)

User Goal: Understand the campaign and start the challenge.

Visual Description

  • Hero Section: Full-screen cinematic background of the MSC World Explora ship at sea.
  • Branding: "MSC World Explora" title in premium, gradient typography.
  • Call to Action: "Take my MSC World Explora Challenge" subtitle with a prominent "Start Challenge" button.
  • Social Proof: A "Curated by Brett Conti" badge adds a personal influencer touch.
  • Incentive: Clear text indicating "Earn total £10 MSC loyalty rewards" for completing the challenge.

2. Teleport Station (Hub)

User Goal: View voyage details and select an experience to teleport to.

Visual Description

  • Atmosphere: Sci-fi inspired "Teleport" interface with a countdown timer "CONNECTION IN 60s".
  • Left Panel (Voyage Details): A form capturing user intent:
    • "Have you already booked?" (Yes/No toggle).
    • "Sailing Date" (Optional).
    • "Special Occasion" (Birthday, Anniversary, etc.).
  • Right Panel (Experiences): A grid of available locations to visit.

Visual Reference (Experience Card Style)

(Users select from cards similar to this style)

3. Experience: Balcony Room

User Goal: Explore the luxury accommodation and find the first clue.

Key Interaction

  • User looks around the 3D room.
  • Objective: Find the "Magic Ring" or interaction point.
  • Audio guide: Voiceover describes the view and comfort.

Visual Reference

4. Experience: Thermal Spa

User Goal: Experience the wellness facilities.

Key Interaction

  • Atmosphere: Relaxing steam and warm lighting.
  • Objective: Locate the hidden coin to unlock the "Thermal Spa Haven" badge.
  • Reward: "Time to catch a vibe" voiceover.

Visual Reference

5. Experience: Hola Tacos & Cantina

User Goal: Explore dining options.

Key Interaction

  • Atmosphere: Vibrant, colorful restaurant setting.
  • Objective: Find the hidden taco or coin.
  • Context: "Authentic Mexican food" highlighted in the description.

Visual Reference

6. Ranking Page

User Goal: Rank their favorite experiences to personalize their potential itinerary.

Visual Description

  • Interface: A drag-and-drop or selection list of the visited locations.
  • Instruction: "Rank Your Experiences - Select the experiences you want to see first".
  • Action: Users click on the 5 experience cards to order them (1 to 5).
  • Completion: "Start Challenge" (or "Finish") button appears after ranking.

7. Completion & Rewards

User Goal: Receive rewards and call-to-action for booking.

Visual Description

  • Success State: "Challenge Completed!" large animated text.
  • Reward: Display of "£10 MSC REWARDS EARNED".
  • Personalized Offer: Based on their rankings/choices:
    • If they liked the Balcony -> "Upgrade to Yacht Club".
    • If they liked the Spa -> "Get a Thermal Spa Pass".
  • CTA: "BOOK YOUR JOURNEY NOW" or "UPGRADE YOUR SUITE".
